# QA Technical Assessment

## Application
SauceDemo

O sistema foi escolhido por ser um ambiente estável e amplamente utilizado para demonstrações de automação de testes, permitindo foco na estratégia de testes, cobertura funcional e implementação do pipeline CI/CD.

## Scope
- Login válido
- Login inválido
- Adição de produto ao carrinho

## Deliverables
- Histórias de usuário
- Critérios de aceite
- Casos de teste
- Estimativa
- Testes automatizados Cypress
- Pipeline GitHub Actions

## How to execute

npm install

npx cypress open

ou

npx cypress run
